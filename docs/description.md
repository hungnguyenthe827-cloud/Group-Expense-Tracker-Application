# Group Expense Tracker Application

## 1. Project Overview
The Group Expense Tracker Application is a web-based system designed to help users manage shared expenses in a simple and transparent way. This application is especially useful for groups of people who share costs, such as friends traveling together, roommates sharing rent and utilities, or students organizing group activities.

Instead of manually calculating expenses, the system allows users to record payments, split costs among group members, and automatically calculate balances. It also provides a clear overview of who owes whom and how much, reducing confusion and potential conflicts.

---

## 2. Problem Statement
Managing shared expenses manually can lead to several issues:

- Difficulty tracking who paid for what  
- Incorrect calculations when splitting bills  
- Lack of transparency in group spending  
- Time-consuming manual calculations  

For example, when one person pays for a group expense, others may forget or miscalculate how much they owe. Without a proper system, this can cause misunderstandings among group members.

This project aims to solve these problems by providing an automated and user-friendly expense tracking system.

---

## 3. Project Objectives
The main objectives of this project are:

- To develop a web application for managing group expenses  
- To automatically calculate how expenses are split among users  
- To provide a clear and transparent view of financial balances  
- To simplify the settlement process between users  
- To improve efficiency and reduce errors in expense management  

---

## 4. Target Users
This application is intended for:

- Groups of friends traveling together  
- Roommates sharing living expenses  
- Students working on group projects or events  
- Small teams managing shared costs  

---

## 5. Key Features

### 5.1 User Authentication
- User registration and login  
- Secure access to personal and group data  

### 5.2 Group Management
- Create a group  
- Add or remove members  
- View group information  

### 5.3 Expense Management
- Add new expenses  
- Record who paid for the expense  
- Select participants involved in the expense  

### 5.4 Expense Splitting
- Split expenses equally among members  
- Support flexible splitting (optional extension)  

### 5.5 Balance Calculation
- Automatically calculate how much each member owes  
- Display clear balances for all users  

### 5.6 Settlement
- Show simplified transactions (who pays whom)  
- Reduce the number of payments needed  

---

## 6. Technology Stack

The system will be developed using the following technologies:

- Backend: Java Spring Boot
